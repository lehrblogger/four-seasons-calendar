Goal: Convert my Four Seasons Resort itineraries into calendar files.

Project Plan:
 - Reverse-engineer the FS website API for trip itineraries.
 - Generate .ics files from the JSON.
 - Store the .ics files in a git repository so that it's easy to audit changes.
 - Use the .ics files on a Radicale server so that I can view them alongside my other calends on macOS and iOS.
